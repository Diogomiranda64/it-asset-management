# IT Asset Management System

## Description

IT Asset Management System developed to centralize and organize the company's technological resources. It enables equipment management, maintenance tracking, historical records consultation, report generation, and improved asset traceability through QR code integration.

## Features

- User authentication
- Equipment management
- Asset categorization
- Maintenance history
- QR Code generation
- Dashboard and statistics
- PDF reports

## Technologies

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
